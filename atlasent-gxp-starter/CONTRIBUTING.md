# Contributing

We welcome contributions from the compliance and life sciences community.

## What We Accept

- **New integration examples** — Wrappers for AI agent frameworks not yet covered
- **New policy templates** — Starter policies for regulations not yet included (e.g., MHRA, Health Canada, PMDA)
- **Improvements** — Corrections to regulatory references, additional context fields, clearer guidance

## Submission Process

1. Fork this repository and create a branch from `main`
2. Follow the structure and conventions of existing files
3. Submit a pull request with a clear description of what you're adding and why

## Guidelines

- Policy JSON files must be valid and follow the established schema
- Use realistic field values that reflect regulated operations
- Regulatory references must cite specific sections or articles
- Keep language professional and compliance-oriented
